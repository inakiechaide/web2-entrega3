Endpoints disponibles:
Clientes:

GET http://localhost/web2-entrega3/api/clientes/read.php - Lista todos los clientes
GET http://localhost/web2-entrega3/api/clientes/read_one.php?id=1 - Obtiene un cliente específico
POST http://localhost/web2-entrega3/api/clientes/create.php - Crea un nuevo cliente
PUT http://localhost/web2-entrega3/api/clientes/update.php - Actualiza un cliente


Turnos:

GET http://localhost/web2-entrega3/api/turnos/listar.php - Lista todos los turnos
GET http://localhost/web2-entrega3/api/turnos/leer_uno.php?id=1 - Obtiene un turno específico
POST http://localhost/web2-entrega3/api/turnos/crear.php - Crea un nuevo turno
PUT http://localhost/web2-entrega3/api/turnos/actualizar.php - Actualiza un turno
DELETE http://localhost/web2-entrega3/api/turnos/eliminar.php?id={id_turno}- Elimina un turno 

GET /api/turnos/listar.php?id_cliente=1&finalizado=0 - Turnos pendientes de cliente especifico


# Filtrar y ordenar
GET /api/turnos/listar.php?fecha_desde=2024-04-01&finalizado=0&orden=fecha_turno,hora_turno&direccion=DESC,ASC

# Paginar
GET /api/turnos/listar.php?pagina=2&por_pagina=15

# Combinar todo
GET /api/turnos/listar.php?fecha_desde=2024-04-01&finalizado=0&orden=fecha_turno&direccion=DESC&pagina=2&por_pagina=15
GET http://localhost/web2-entrega3/api/turnos/listar.php?fecha_desde=2024-04-01&fecha_hasta=2024-04-30&orden=fecha_turno,hora_turno&direccion=DESC,ASC&pagina=1&por_pagina=15 - Listado y ordenamiento de turnos por fecha, hora, dirección(DESC - ASC) y paginación
